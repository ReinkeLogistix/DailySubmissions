Expected CSV format (first row must be headers)

Common column: action (add | update | delete)

Drivers:
action, driver_code, driver_name, pin_hash, is_active, default_shift_code

Trucks:
action, truck_code, registration, division, is_active

Trailers:
action, trailer_code, registration, is_active

Shifts:
action, shift_code, shift_name, is_active

Notes:

  • action = add → creates new record (fails if duplicate code exists)
  • action = update → updates existing record (must already exist)
  • action = delete → removes record (must exist)
  • Codes must be unique: driver_code, truck_code, trailer_code, shift_code
  • is_active: true / false

Notes

Admin page supports add, update, and delete via the CSV action column.
Duplicate codes are not allowed. Matching codes are updated only when action = update.
Use: driver_code, truck_code, trailer_code, or shift_code as the key reference.
